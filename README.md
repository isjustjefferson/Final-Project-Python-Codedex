# Final Project: Codédex Python

Este é o projeto final da trilha "The Legend of Python" da plataforma **Codédex**. O objetivo deste desafio foi aplicar e consolidar os fundamentos da linguagem para construir uma solução lógica e bem estruturada.

---

## 🇧🇷 Português

###  Descrição
O projeto marca a conclusão da jornada de Python na Codédex. Mesmo já tendo experiência prévia com a linguagem em outros projetos acadêmicos, utilizei este desafio para revisar conceitos, eliminar lacunas e consolidar uma base extremamente sólida para o desenvolvimento de soluções robustas no ecossistema de Backend.

### Tecnologias Utilizadas
* **Python 3**: Linguagem principal para o desenvolvimento da lógica.
* **TKinter**: Biblioteca em Python para interface gráfica.
  
### O que este projeto demonstra:
* **Maturidade Técnica**: Revisão e aplicação refinada de conceitos fundamentais.
* **Controle de Fluxo e Dados**: Uso eficiente de listas, dicionários, condicionais e laços de repetição.
* **Princípio DRY (Don't Repeat Yourself)**: Foco em escrever código limpo e reaproveitável através de funções.
* **Base para Backend**: Domínio da sintaxe essencial para atuar com frameworks como Django ou Flask, ou em projetos de análise de dados.
